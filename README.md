# 🛒 쇼핑 리스트 앱

간단하고 깔끔한 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료 처리
- 개별 아이템 삭제
- 완료된 항목 일괄 삭제
- localStorage로 데이터 자동 저장 (새로고침해도 유지)
- 완료 진행률 표시 (N / 전체 완료)

## 사용법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

별도 설치나 서버 없이 동작합니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
